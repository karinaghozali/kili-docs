    Prerequisites

    Install Kili Playground:

    git clone https://github.com/kili-technology/kili-playground.git
cd kili-playground

    and authenticate in python (see /docs/authentication):

    client, user_id = authenticate('MY_EMAIL', 'MY_ADDRESS')

    See the Honeypot of an annotation

    The honeypot is an attribute of the Asset object. We can therefore use the export functions of the Assets to retrieve the honeypots:

    export_assets(client, project_id='d9729feb74992cc3482b350163a1a010')

get_assets_by_external_id(client, project_id, external_id)
