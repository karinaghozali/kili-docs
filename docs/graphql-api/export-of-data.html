## Export data

Kili has two ways to export data:

## Bulk

Good for workflows that require access to all labels in a project.

"Python
from kili.queries.asset import export_assets

export_assets(client, project_id='d9729feb74992cc3482b350163a1a010')

    Asset by asset

    Good for real-time use cases or for exporting a subset of data:

    "Python from kili.queries.asset import get_assets_by_external_id

    get_assets_by_external_id(client, project_id, external_id)
