    Create a user

    Prerequisites

    Install Kili Playground:

    git clone https://github.com/kili-technology/kili-playground.git
cd kili-playground

    and authenticate in python (see/docs/authentication):

    client, user_id = authenticate('MY_EMAIL', 'MY_ADDRESS')

    Create a user

    Users are members of a Kili Project. They belong to an Organization.

    from kili.mutations.user import create_user

create_user(client, name, email, password, phone, organization_id, organization_role)

    The role in the organization can be: ADMIN or USER.
