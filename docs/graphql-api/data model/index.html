Kili Docs

    Data model

    User: it is the account of a person attached to an organization

    Organization: this is the company to which the users.

    Project: it is the combination of an interface, a dataset, a participants, quality parameters, and potentially a model

    ProjectUser: it is the participant in a project with a defined role

    Tool: it is a component of the interface. The interface is composed of several tools.

    Asset: this is the image or document to be annotated

    Label: it is the set of annotations made on an asset.
