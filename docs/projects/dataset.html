    View assets

    Thanks to "Assets", we can visualize the list of assets, priority (for active learning), annotators who have worked on asset annotation, status, and metrics (duration, consensus, honeypot) at the asset level. From here, you can also search your assets.

    Search assets

    To dive into your assets, you can use the search bar. To combine filters, just add a space between to terms. For instance: status:labeled consensus<80.

    • External ID = externalId:image 1.png, image 2.png
    • Creation Date = not searchable
    • Priority = not searchable
    • Labeled by = labeledBy:John Doe, Jane Doe
    • Duration = not searchable
    • Status = status:todo | status:ongoing | status:labeled | status:reviewed
    • Honeypot = honeypot<10 | honeypot>10
    • Consensus = consensus<90 | consensus>90

    View labels

    "Labels" submenu allows you to view labels:

    • label type (prediction if made by a model, default if made by a human),
    • annotators,
    • time spent annotating,
    • status of the asset, and
    • quality metrics.

    From here, you can also search your labels.

    Search labels

    To dive into your labels, you can use the search bar. To combine filters, just add a space between to terms. For instance "status:labeled consensus<80"

    • External ID = externalId:image 1.png, image2.png
    • Creation Date = creationDate>25/11/2019 | creationDate<25/11/2019
    • Type = type:prediction | type:default | type:review
    • Labeled by = labeledBy:XXX
    • Duration = not searchable
    • Status = status:todo | status:ongoing | status:labeled | status:reviewed
    • Honeypot = labelHoneypot<10 | labelHoneypot>10
    • Consensus = consensus<90 | consensus>90
    • Skipped = skipped:true | skipped:false

    Asset processing workflow

    There are 4 status

    • TODO: the asset is to be processed

    • ONGOING: the asset is being processed by an annotator, or when there is a consensus, awaiting processing by others annotators.

    • LABELED: the asset has been annotated by an annotator, or when there is a consensus, by the different annotators.

    • REVIEWED: at least one of the labels of an asset has gone through the Review process.
