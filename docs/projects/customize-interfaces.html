    Customize interfaces

    Interfaces can be customized in JSON settings as explained in Settings section.

    Interfaces

    Almost all interfaces (excepted custom interfaces) are divided in two parts:

    • the asset viewer on the left
    • the jobs viewer on the right

    Jobs are implementation of machine learning tasks (such as classification, object detection, etc). Interfaces can chain several jobs one after the other. Each job can be either required or optional.

    JSON settings

    The JSON settings fully describe an interface. In order to chain 3 jobs for instance, it is composed as such:

    {
  "filetype": "IMAGE" | "TEXT" | "VIDEO",
  "jobs": {
    "job_1": {
      "mlTask": "CLASSIFICATION",
      "instruction": "Are grapes fully grown?",
      "content": {
        "categories": {
          "YES": {
            "name": "Yes",
            "children": []
          },
          "NO": {
            "name": "No",
            "children": []
          }
        },
        "input": "radio"
      },
      "required": 0 | 1 (whether the job is optional or required)
    },
    "job_2": {
      "mlTask": "CLASSIFICATION",
      "instruction": "Which types of grapes are there?",
      "content": {
        "categories": {
          "PINOT_NOIR": {
            "name": "Pinot Noir",
            "children": []
          },
          "RIESLING": {
            "name": "Riesling",
            "children": []
          }
        },
        "input": "checkbox"
      },
      "required": 0 | 1 (whether the job is optional or required)
    },
    "job_3": {
      "mlTask": "OBJECT_DETECTION",
      "tools": [
        "rectangle",
        "polygon"
      ],
      "instruction": "Object detection",
      "content": {
        "categories": {
          "ROTTEN": {
            "name": "Rotten grape",
            "children": []
          },
          "CLEAN": {
            "name": "Clean grape",
            "children": []
          }
        },
        "input": "radio" | "checkbox"
      },
      "required": 0 | 1 (whether the job is optional or required)
    }
  }
}
