    Create a project

    Prerequisites

    Install Kili Playground:

    pip install kili

    and authenticate in python (see Authenticate):

    from kili.authentication import KiliAuth
from kili.playground import Playground

kauth = KiliAuth(email=email, password=password)
playground = Playground(kauth)

    Create a project

    To create a project, you need the following information:

    • project title
    • project description
    • project type
    • whether or not the project should use a honeypot
    • JSON describing the interface (jsonInterface)
    • a list of users you want to give access to the project

    Here is an end-to-end python example of how to configure a new project in Kili:

    title = 'Test'
description = 'Ceci est un projet test'
json_interface = {
    "filetype": "IMAGE",
    "jobs": {
        "JOB_0": {
            "mlTask": "OBJECT_DETECTION",
            "tools": ["rectangle"],
            "instruction": "Categories",
            "required": 1,
            "isChild": False,
            "content": {
                "categories": {
                    "OBJECT_A": {
                        "name": "Object A",
                        "children": ["JOB_1"]
                    },
                    "OBJECT_B": {
                        "name": "Object B",
                        "children": ["JOB_2"]
                    }
                },
                "input": "radio"
            }
        },
        "JOB_1": {
            "mlTask": "TRANSCRIPTION",
            "instruction": "Transcription of A",
            "required": 1,
            "isChild": True
        },
        "JOB_2": {
            "mlTask": "TRANSCRIPTION",
            "instruction": "Transcription of B",
            "required": 1,
            "isChild": True
        }
    }
}

project = playground.create_empty_project(user_id=playground.auth.user_id)
playground.update_properties_in_project(project_id=project['id'],
                                        title=title,
                                        description=description,
                                        json_interface=json_interface)

# Here you can iterate on the other users you want to add to the project
for email in emails:
    playground.append_to_roles(
        project_id=project['id'], user_email=email, role='LABELER')
