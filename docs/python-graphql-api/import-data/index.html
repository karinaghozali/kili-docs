    Importing data

    Prerequisites

    Install Kili Playground:

    pip install kili

    and authenticate in python (see Authenticate):

    from kili.authentication import KiliAuth
from kili.playground import Playground

kauth = KiliAuth(email=email, password=password)
playground = Playground(kauth)

    Import data

    NLP Project

    The content of the documents is directly imported in text format:

    assets = playground.create_assets(project_id='d9729feb74992cc3482b350163a1a010',
  contents=['Hello world!', 'This is another hello world.'],
  external_ids=['doc_1', 'doc_2'])

    Computer Vision Project

    If your data is in image format, it must be hosted on a web server (for example Google Drive), then you point Kili to your data by giving the API the URLs to each file.

    assets = playground.create_assets(project_id='d9729feb74992cc3482b350163a1a010',
  contents=['https://drive.google.com/uc?export=view&id=17uAKNmBuojnXwTfts-HveAcFNXzBtv6i'],
  external_ids=['image_1'])
