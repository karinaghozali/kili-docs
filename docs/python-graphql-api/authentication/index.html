Kili Docs

    Authentication

    Prerequisites

    Install Kili Playground:

    git clone https://github.com/kili-technology/kili-playground.git
cd kili-playground

    Authentication

    The Kili API secures all requests and mutations with JWT tokens.

    To use GraphQL, you must:

    • Authenticate via the signIn resolver, in order to retrieve an authentication token. Your password can be omitted if you pass it in the KILI_USER_PASSWORD environment variable:
    from kili.authentication import authenticate

client, user_id = authenticate('john.doe@facebook.com')
    • Initiate the desired transfer/request (see next sections):
    Data modification actionFunction signature
    Create a usercreate_user(client, name, email, password, phone, organization_id, organization_role)
