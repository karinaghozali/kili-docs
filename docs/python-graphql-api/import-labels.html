    Import labels

    Prerequisites

    Install Kili Playground:

    pip install kili

    and authenticate in python (see Authenticate):

    from kili.authentication import KiliAuth
from kili.playground import Playground

kauth = KiliAuth(email=email, password=password)
playground = Playground(kauth)

    Import labels

    From the API you can import two types of labels:

    • honeypot labels to calculate the performance of an annotator in a supervised way on a task
    • prediction type labels to label (correct) the output of a model

    To create a new label for a data line, we will need:

    • The ID of the Asset to which the Label relates
    • A Python object json_response describes the annotations of the asset in question (see below)
    playground.create_honeypot(asset_id, json_response)
playground.create_prediction(asset_id, json_response)
