Kili Docs

    Create a project

    Prerequisites

    Install Kili Playground:

    git clone https://github.com/kili-technology/kili-playground.git
cd kili-playground

    and authenticate in python (see/docs/authentication):

    client, user_id = authenticate('MY_EMAIL', 'MY_ADDRESS')

    Create a project

    To create a project, you need the following information:

    • project title
    • project description
    • project type
    • whether or not the project should use a honeypot
    • json as a string describing the interface

    Here is an end-to-end python example of how to configure a new project in Kili:

    from kili.mutations.project import create_project

create_project(client,
  title='Project Title',
  description='',
  tool_type='IMAGE',
  use_honeypot=True,
  interface_json_settings="{ \\\"annotation_types\\\": { \\\"GRAPE\\\": \\\"Grape\\\", \\\"LEAF\\\": \\\"Leaf\\\"} }")

    toolType can take the following values:

    CLASSIFICATION
SENTIMENT
NER
RELATION_EXTRACTION
ENTITY_GROUPING
COREFERENCE_RESOLUTION
TRANSLATION
TRANSCRIPTION
IMAGE
MULTICLASS_IMAGE
CUSTOM
