Kili Docs

    Image

    Kili Technology provides an image annotation module with all the segmentation and classification tools necessary to carry out the most common image annotation tasks.

    These segmentation and classification tools are designed to be combined in the same interface.

    Image segmentation tools

    • Rectangle or 2D bounding box

    • Polygon

    • Semantic (pixel-wise)

    • Line

    • Point

    Image classification tools

    • Simple choice

    • Multiple choice

    • Class hierarchy
