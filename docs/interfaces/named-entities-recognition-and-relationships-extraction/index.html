    Named entities recognition & relationships extraction

    Kili Technology fournit une interface d'annotation de Reconnaissance d'entités nommées (NER) et d'annotation des relations entre entités pour réaliser les tâches d'annotation de NER les plus courantes.

    NER

    Une fois la liste des classes d'entités configurée, l'annotation est simple. La sélection du texte amène l'affichage de la liste de types d'entitées, sur laquelle il suffit de cliquer.

    Relationship extraction

    Au clic sur « Link Entities », il suffit de préciser le nom du groupe et de cliquer sur les entités que vous souhaitez relier. Lorsque le groupement est fini, de cliquer « Stop Linking ».
