    Ajouter un utilisateur à un projet

    Pré-requis

    Installer Kili Playground:

    git clone https://github.com/kili-technology/kili-playground.git
cd kili-playground

    et s'authentifier en python (voir /docs/authentification):

    client, user_id = authenticate('MY_EMAIL', 'MY_ADDRESS')

    Ajouter un utilisateur à un projet

    Pour accorder l'accès à un projet à un utilisateur qui fait déjà partie de votre organisation, vous devez obtenir l'ID du projet et du rôle ainsi que l'adresse e-mail de l'utilisateur:

    append_to_roles(client, project_id, user_email, role)

    Le rôle sur un projet peut être: ADMIN, REVIEWER, LABELER ou READER.
