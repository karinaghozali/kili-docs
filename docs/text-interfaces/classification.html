    Text classification

    Kili Technology provides a simple classification interface for text files.

    Single choice

    For single choice, choose input: radio button.

    Multiple choice

    For multiple choice, choose input: check box.

    Dropdown

    For dropdown choice, choose input: dropdown. Dropdown is especially relevant for long lists. It allows to search, and to filter out easily the list.

    Nested classification

    To build nested classification, choose input: check box. See configuration in customize-interface section.
