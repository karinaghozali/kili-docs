    Donnée on premise ou sur cloud privé

    Kili Technology fonctionne avec de la donnée hébergée On Premise ou sur un cloud privé. La donnée source est accessible directement depuis l'ordinateur client et n'est jamais partagée (ou accessible) par Kili Technology.

    Cela signifie que Kili Technology n'aura accès à aucune donnée source (images, texte, etc...), et Kili Technology ne stockera que les annotations et un ID vous permettant de faire le lien avec la donnée source.

    Donnée sur Cloud Privé avec VPN/VPC

    Pour permettre à des équipes de travailleurs distants d'accéder à la donnée On Premise ou sur Cloud Privé, utilisez un VPN. Voici quelques tutoriels sur la mise en place d'un VPN pour les principaux opérateurs de cloud:

    https://cloud.google.com/vpn/docs/how-to/choosing-a-vpn

    https://aws.amazon.com/blogs/aws/new-vpc-endpoint-for-amazon-s3/

    https://azure.microsoft.com/en-us/services/vpn-gateway/

    Créer un fichier CSV avec les URLs

    Si votre donnée est hébergée dans un cloud (par exemple Amazon S3), vous pouvez charger votre donnée dans Kili Technology en créant un fichier CSV avec des URL vers chaque fichier.

    Le format du fichier devra avoir deux colonnes : External ID, URL

    externalId,url
image_1,https://images.pexels.com/photos/45209/purple-grapes-vineyard-napa-valley-napa-vineyard-45209.jpeg
image_2,https://images.pexels.com/photos/760281/pexels-photo-760281.jpeg

    Sélectionnez ensuite le fichier CSV sur votre ordinateur afin de le charger.

    Donnée On Premise

    Voici un guide pour utiliser Kili Technology avec les données sur votre disque dur pour Mac OSX et Linux. Il faut d'abord démarrer un serveur HTTP local pour servir les fichiers, puis générer un CSV de liens vers les fichiers et les télécharger vers Kili Technology.

    Placer tous les fichiers dans un seul dossier

    Placez tous les fichiers que vous voulez étiqueter dans un seul dossier sur votre disque dur.

    Obtenir l'adresse IP de votre ordinateur

    La commande ci-dessous devrait donner votre adresse IP, ex : 192.168.1.112

    ifconfig en0 \| grep inet \| grep -v inet6 \| awk\'{print \$2}\'\'

    Démarrez le serveur HTTP sur votre ordinateur

    Vous pouvez démarrer un serveur local via la ligne de commande.

    Avec Python

    python -m SimpleHTTPServer

    Avec NodeJS

    npm install -g http-server; http-server -p 8000

    *Notez dans cet exemple que le port à servir est le 8000.

    Créer un CSV avec vos données

    Si vous allez sur vous devriez voir la liste de répertoires avec vos fichiers.

    Faites cd dans le répertoire avec tous vos fichiers et exécutez la commande ci-dessous qui va générer data.csv

    IP_ADDRESS=\$(ifconfig en0 \| grep inet \| grep -v inet6 \| awk
\'{print \$2}\')

CSV=\$(echo \"Data URL\"; for fileName in \$(ls); do echo
"\$fileName,http://\$IP\_ADDRESS:8000/\$fileName"; done)

    Charger data.csv

    Dans l'onglet Dataset, cliquez sur Add New

    Voir ci-dessus pour des informations détaillées sur le format CSV que Kili Technology attend.

    Notes

    Seuls les utilisateurs d'un même réseau peuvent voir vos données si vous les hébergez vous même.

    Si le serveur local est arrêté, vous perdrez l'accès à vos données pendant que vous utilisez Kili Technology.

    Pour donner accès à la donnée à des utilisateurs extérieurs, il faut exposer votre Localhost sur Internet. Voici un tutoriel pour le faire rapidement https://dev.to/levivm/exposing-localhost-server-to-the-internet-in-one-minute-2713
