    How to accelerate annotation with machine learning

    Kili Technology allows to use machine learning to speed up annotation project. You can:

    • Import labels which can be

      • predictions from your custom model
      • predictions from a weakly supervised learning framework
      • human labeled data from previous project or other sources

    • Orchestrate online learning with an autoML framework

    • Implement active learning strategies with queue prioritisation

    Import labels

    Kili Technology allows to import labels, so that annotators work on preannotated assets. In such case, the task is to check predictions and to amend it, in order to match expected quality.

    This can be used to speedup annotation process or to run quality check task. Experience shows that add value of such approach vary a lot depending on task nature, complexity, and predictions accuracy.

    Predictions from your custom model

    Once inference phase is done on your dataset, you can upload predictions using this recipe on Kili-Playground.

    Predictions from a weakly supervised learning framework

    Weakly supervised learning maturity depends on tasks. Our experience shows that is can be extremely powerfull on text annotation, classification and Named Entities Recognition tasks.

    Weak supervision is the fact to combined weak predictors in order to build a more robsut one, for instance:

    • Hard-coded heuristics: usually regular expressions (regexes)
    • Syntactics: for instance, Spacy’s dependency trees
    • Distant supervision: external knowledge bases
    • Noisy manual labels: crowdsourcing
    • External models: other models with useful signals

    To know a bit more about weak supervision, here.

    We are use to work with Snorkel, a framework create at Standford.

    Human labeled data

    For a variety of good reasons you could need to review/reannotate human labeled data. For instance: reviewing or re annotating an annotated dataset sourced outside, making a quality check on an already annotated datasets, labeling the human generated logs from a chat bot framework...
