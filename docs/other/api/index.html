    Getting started

    Here is a list of API function that you can access from our endpoint:

    https://cloud.kili-technology.com/api/label/graphql

    Mutations Resolvers

    Mutations are used to insert data into Kili database or alter data already in Kili database.

    CreateOrganization

    Here is an example how to create an organization:

    mutation {
  createOrganization(
    name: "Example Corp"
    address: "1 avenue des Champs Elysées"
    zipCode: "75008"
    city: "Paris"
    country: "France"
  ) {
    id
  }
}

    CreateUser

    Here is an example:

    mutation {
    createUser(
        name: "John Smith"
        email: "john.smith@example.com"
        password: "******"
        phone: ""
        organizationID: "Example Corp"
        organizationRole: USER
    ) {
        id
    }
}

    CreateProject

    Here is an example:

    mutation{
    createProject(
        userID: "user-id-returned-by-previous-mutation"
    )
    {
        id
    }
}

    UpdateProject

    Here is an example:

    mutation {
    updateProject(
        projectID: "project-id-returned-by-previous-mutation"
        title: "My Project"
        description: "This is a beautiful project"
        creationActiveStep: 0
        creationCompleted: [0, 1, 2, 3, 4, 5, 6]
        creationSkipped: []
        interfaceCategory: {{ project_interface_category }}
        inputType: TEXT
        interfaceTitle: null
        interfaceDescription: null
        interfaceUrl: null
        outsource: false
        useConsensus: false
        consensusTotCoverage: 0
        minConsensusSize: 0
        maxWorkerCount: 4
        minAgreement: 66
        useHoneyPot: false
        instructions: null
        modelTitle: ""
        modelDescription: ""
        modelUrl: ""
    )
    {
        id
    }
}

    Query Resolvers

    GetProjects

    Here is an example:

    query {
    getProjects(
        userID: "user-id"
    )

        id
    }
